Projects

2016


GTA: GSMem Two-factor Authentication

Course project in which we implemented a 2-factor-authentication scheme which requires absolutely no user interaction. To accomplish this, we use the electromagnetic radiation emitted by a machine's memory bus lines to authenticate with a user's cell phone.


Startup Project: Data Exfiltration from Airgapped Machines

My first task as a Ph.D. student was to spend a month reproducing the results of a recent paper. I chose GSMem, a paper from Usenix Security Symposium 2015 in which the authors demonstrate how to establish a covert channel on top of the electromagnetic radiation emitted by a machine's memory bus lines, and thus exfiltrate data from an airgapped machine.
